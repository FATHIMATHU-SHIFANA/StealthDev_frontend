# Stealth Dev Frontend

Frontend client for Stealth Dev (React + Vite + TypeScript + PWA).

For complete project documentation (frontend + backend + architecture + API + troubleshooting), read:

- `../README.md`

## Quick Start

```bash
npm install
npm run dev
```

App runs at `http://localhost:3000` by default.

## Environment

Create `.env`:

```env
VITE_SOCKET_URL=http://localhost:5000
```

## Scripts

- `npm run dev` - start dev server
- `npm run build` - build production bundle
- `npm run preview` - preview production build
- `npm run lint` - run lints
